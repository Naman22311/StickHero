1. Assets Folder:
   - Contains all assets used in creating the UI skeleton screens.

2. UI Folder:
   - Houses UI skeleton screens crafted through FXML files.

3. FXMLFilesUI Folder:
   - Stores all FXML files responsible for generating the UI screens.

4. src -> Screens Folder:
   - Encompasses fundamental skeleton code depicting class structure and their relationships/hierarchy.

5. UML and UseCase Folder:
   - Includes UML diagrams (both light and dark themes) and the use case diagram.

6. Idea.md File:
   - Contains the current project idea, incorporating the application of Object-Oriented Programming (OOP) concepts. Describes classes and their attributes.
